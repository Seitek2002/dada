# Видео файлы

Поместите ваши тестовые видео файлы в эту папку.

Требуемые файлы:
- video1.mp4
- video2.mp4
- video3.mp4
- video4.mp4
- video5.mp4

Рекомендации:
- Формат: MP4
- Разрешение: 1080x1920 (вертикальное)
- Длительность: 15-60 секунд
- Размер: до 50MB

Для тестирования вы можете использовать любые короткие вертикальные видео.
